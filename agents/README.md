# Trained Agents

Stores pre-trained linear FA agents as `.pkl` files generated by `train_base_agent.py`.

Files are named `mnk_{n}x{n}_k{k}_level_{level}.pkl`.

## How to Generate

```bash
cd adaptive_difficulty_rl

# 3×3 Tic-Tac-Toe (k=3)
python rl_training/train_base_agent.py --n 3 --k 3

# 5×5 board, k=4
python rl_training/train_base_agent.py --n 5 --k 4

# 15×15 Gomoku (k=5)
python rl_training/train_base_agent.py --n 15 --k 5 --episodes 200000
```

## How to Load

```python
from rl_training.mnk_q_agent import MNKQLearningAgent

agent = MNKQLearningAgent()
agent.load("agents/mnk_3x3_k3_level_3.pkl")
agent.epsilon = 0.0  # disable exploration for inference
```
